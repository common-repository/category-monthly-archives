This Widget display category monthly archives links on category page.

== Description ==

This Plugin adds `Category Monthly Archives` to Widget.
Added widget displays category monthly archives links on category page.
Only displays links on category page.

== Installation ==

1. Upload `category-monthly-archives` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Add `Category Monthly Archives` to Appearance > Widgets 

== Frequently Asked Questions ==

= Where to links displays ? =

This Widget only display links on category page.
